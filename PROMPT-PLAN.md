# Промпт-план: лендинг клуба единоборств «Самсон»

Одностраничный сайт **без бэкенда**. Стек: React + Vite + Tailwind.  
Визуальный ориентир: энергия [Champ Belts](https://champbelts.ae/ru/) + современная графика с референсов (асимметрия, крупная типографика, зернистость, градиенты). **Не копировать** champbelts — взять логику подачи, но свой визуальный язык.

---

## 1. Стек

| Слой | Технология |
|------|------------|
| UI | React 18 |
| Сборка | Vite 5 |
| Стили | Tailwind CSS 3 + PostCSS |
| Иконки | lucide-react |
| Шрифты | Google Fonts: **Syne** (display) + **DM Sans** (body) |
| Данные | `src/data/*.js` — статика |
| Запись | Telegram / WhatsApp — ссылки с предзаполненным текстом |
| Деплой | GitHub Pages / Netlify / Vercel |

**Без:** API, сервера, БД, форм с POST, CMS.

## 2. Дизайн-система

### Палитра (синтез из 3 референсов)

| Token | HEX | Роль |
|-------|-----|------|
| `ink` | `#1C1C1A` | Основной текст, тёмные блоки |
| `zaffre` | `#0D21A5` | Primary — сила, глубина, hero-градиенты |
| `navy` | `#16336F` | Вторичный тёмно-синий, карточки |
| `periwinkle` | `#B9C7F8` | Светлые фоны секций, мягкие градиенты |
| `citron` | `#D1D067` | Акцент — энергия, бейджи, hover |
| `coral` | `#FE492A` | Hot CTA — «Первое занятие бесплатно» |
| `cream` | `#F0EEDF` | Тёплый фон вместо белого |
| `lime` | `#C8E600` | Доп. акцент для графики (органические blob'ы) |

### Градиенты

```css
/* Hero */
bg-gradient-to-br from-zaffre via-[#1a2d8f] to-navy

/* Светлые секции */
bg-gradient-to-b from-cream to-periwinkle/30

/* Акцентная полоса */
bg-gradient-to-r from-citron to-lime

/* Тёмный блок тренера */
bg-gradient-to-tr from-ink via-navy to-zaffre
```

### Текстуры и графика

- **Grain overlay** — псевдоэлемент `::after` с `opacity: 0.04–0.08`, `mix-blend-mode: overlay`, SVG noise или `radial-gradient` dot pattern (без длинных `data:image` в Tailwind arbitrary classes — только CSS-файл)
- **Organic blobs** — SVG-фигуры в `public/assets/shapes/` (волны, капли) в цветах zaffre + citron + cream
- **Motion blur feel** — на hero-фото `filter` + градиентный оверлей (как референс с бегуном)
- **Крупная типографика** — H1 до `clamp(48px, 10vw, 120px)`, tight line-height
- **Асимметрия** — split hero: слева текст + CTA, справа графика/фото (как Soft Launch, но спортивно)
- **Карточки** — `rounded-3xl`, тонкий border `white/10` на тёмном / `ink/5` на светлом
- **Кнопки** — pill `rounded-full`, ghost + filled; главный CTA — `coral` или `citron` на тёмном

### Типографика

| Элемент | Шрифт | Стиль |
|---------|-------|-------|
| H1, логотип | Syne 800 | UPPERCASE, tracking tight |
| H2 секций | Syne 700 | Mixed case |
| Подзаголовки | DM Sans 500 | `text-lg`, приглушённый |
| Body | DM Sans 400 | `leading-relaxed` |
| Цены / цифры | Syne 700 | Крупно, акцентный цвет |
| Бейдж «Бесплатно» | Syne 600 | `coral` фон, белый текст |

## 6. UI-паттерны по секциям

### Hero (WOW-эффект)

- Full viewport `min-h-[100dvh]`
- Split layout: **60% текст / 40% визуал** на desktop; stack на mobile
- Фон: градиент zaffre → navy + grain + blob SVG
- Фото спортсмена с cutout или сильным градиентным маскированием
- Бейдж `coral`: «Первое занятие — 0 ₽»
- H1: `САМСОН` крупно + подзаголовок «Клуб единоборств»
- 2 CTA: primary «Записаться бесплатно» · secondary «Смотреть направления»

### Directions (как Champ Belts «Направления»)

- Сетка 2×3 на desktop, 1 col mobile
- Карточка: иконка + название + 1 строка описания + hover lift
- Чередование фона: cream / periwinkle/20

### Coach (как «Главный тренер» на Champ Belts)

- Тёмный full-width блок с градиентом
- Крупное фото слева, текст справа
- Список регалий с иконками-check
- Мини-цитата или УТП тренера

### Pricing

- 2 колонки: **Групповые** | **Индивидуальные**
- Карточка «Популярный» — 12 занятий с бейджем
- Sticky-баннер вверху секции: «Первое групповое — бесплатно»
- Кнопка под каждым тарифом → мессенджер

### Camps

- Яркая секция citron/lime акцент
- Календарь: два блока «1–10 июля» и «21–30 июля»
- CTA «Узнать о сборах»
